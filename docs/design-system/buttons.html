Which button should I use?

Quick decision guide. Find the closest match and reuse its token pattern.

I need...Use this patternKey tokens
Primary action (copy/save) Copy Button (purple) purpleButton, purpleButtonBg
Secondary action (settings) Settings Pill settingsPillBorder, settingsPillFill
Tool selection Circular Icon Button toolActiveBg, iconDefault
Mode switch (2 options) Segmented Control toggleActiveBg, toggleBg
Menu item with shortcut Popover Menu Row system label colors
Success/copied confirmation Green state overlay copiedGreenBorder, copiedGreenBg
Annotation note label Note Pill redNoteBg, noteHoverBg
Close/destructive action Red Icon Button closeHoverBg, closeIconHover
Small status indicator pill Setup Small Pill purpleButtonBg, copiedGreenBg
Large success action Setup Complete Button copiedGreenText, copiedGreenBg
