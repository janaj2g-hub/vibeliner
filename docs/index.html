vibeliner
screenshot · annotate · prompt

Communicate your design intent to an LLM.

Vibeliner is a native macOS menu bar app that lets you capture, annotate, and package screenshots for AI coding tools.

See a bug? Capture it → annotate it → paste directly into Claude, ChatGPT, or Cursor.

Download for macOS
macOS 14+ · Free

how it works

1

Capture

Press ⌘⇧6 to screenshot any region. Purple crosshair, precise selection, branded overlay.

2

Annotate

Drop numbered pins, arrows, rectangles, circles, and freehand marks. Add notes to each.

3

Prompt

Copy the auto-generated prompt and paste it into your AI tool. The LLM sees what you see.
